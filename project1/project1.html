Describe how search engines work (minimum 50 words).

There are 3 different types of search engines:

  1. Crawler
  2. Human
  3. Mutant

The Crawler engine use automated system to "crawl" the web and document everything that it finds.

The Human engine uses actual humans to view, review, and rank tht websites. They then will enter the information they've collected into a database.

The Mutant engine (also referred to as the Hybrid engine) uses crawlers to compile data, then use human input to influence the ranking of each website.

Explain the importance of keywords and descriptions as meta tags (minimum 50 words).

Keywords are the building blocks of language and of search. As the crawler engines document everything on the page, they keep track of the keywords that they find instead of storing all the data. It’s more efficient to store certain keywords in one database then to overload it with everything at once.

Keywords are most important during search because it helps the user sort through the search engine to find specifically what they’re looking for. Keywords dominate our search by matching the keywords to the pages and displaying it to us for easy access.

Meta descriptions are also equally important as keywords. Meta descriptions will give the user a general description of the page thatthey are searching before they enter the website. The main function of the meta description is toadvertise the website and draw usersto the site as a result.

List the four elements that comprise writing relevant content links (minimum 50 words).

Links are comprised of four components:

  1. Starting link tag
  2. Link referral location
  3. Visible / Anchor text of link
  4. Closure link tag

The links and their components are important because the crawlers need to be able to navigate through the pathways of a website to make sure it is the best fit for the users search.

What constitutes good heading hierarchy and include an example with more than one heading tag (minimum 50 words).

Good heading hierarchy is followed when a page begins with the H1 heading and moves down to the H6 heading in sequential order. This mean that H1 will be the main heading, H2 will be the subheading, and so on. By doing this, the headings for each page will be logically placed.

Here is an example:

<body>

  <article>

    <h2>Main Heading</h2>

    <h3>Subheading</h3>

    <h4>Second Subheading</h4>

    <h4>Article Content</h4>

  </article>

</body>

In this example we have a decending header hierarchy with H1 being the main heading of the article, H2 being the subheading of the article, and so on as displayed.
